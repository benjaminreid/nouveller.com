WordPress from scratch : Part 1

Posted on July 15, 2009

Welcome to part one of my 'WordPress from scratch' series! This first part covers downloading and installing a localhost called MAMP, downloading WordPress and setting up a database for it and finally, installing WordPress itself. This time around I'll be doing screen-casts as opposed to saying do this do that, so you should be able to follow me on screen. It would be great to get some feedback on how you think I presented this as well as general feedback from the video, so please leave any comments.
